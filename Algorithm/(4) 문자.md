## 패턴 매칭

### Brute Force
- 본문 문자열을 처음부터 끝까지 차례대로 순회하면서 패턴 내의 문자들을 일일이 비교하는 방식을 동작

```python
p = 'is'                #  찾을 패턴
t = 'This is a book~!'  #  전체 텍스트
M = len(p)
N = len(t)

def BruteForce(p, t):
    i = 0   # t의 인덱스
    j = 0   # p의 인덱스
    while j < M and i < N:
        if t[i] != p[j]:
            i = i - j
            j = -1
        i = i + 1
        j = j + 1
    if j == M : return i - M    # 검색 성공
    else : return -1            # 검색 실패

```
